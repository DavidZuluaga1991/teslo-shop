# TesloDB

1. Clonar proyecto
2. ``` npm i ```
3. Clonar el archivo ```.env.template``` y renombrarlo a ```.env```
4. Camvbiar las variables de entorno
5. Levantar la base de datos

```
docker-compose -d
```
6. Levanta: ```npm run start:dev```
